# Webhook Management App 项目介绍

基于 Svelte 构建的 Webhook 管理系统，提供简洁响应式的用户界面。

## 主要功能

- 创建、编辑和删除 Webhook
- 启用/禁用 Webhook 状态
- 自定义数据触发 Webhook
- 亮色/暗色主题支持
- 响应式三列布局

## 安装指南

```bash
# 克隆仓库
git clone https://github.com/keiSenFront/webhook-management.git

# 安装依赖
yarn install

# 启动开发服务器
yarn dev
```

## 生产环境构建

yarn build

## 技术栈

- Svelte 3
- JavaScript ES6+
- CSS Grid 布局
- Vite 构建工具
